How It Works

From product page to confident checkout.

FitLayer turns a shopper photo and your product imagery into a realistic try-on experience without interrupting the buying journey.

Step 01

Add FitLayer to your store

Connect your storefront, upload your garments, and match the widget to your brand.

Step 02

Shopper uploads a photo

A simple, guided upload begins directly from the product page in just a few seconds.

Step 03

AI creates the try-on

FitLayer generates a realistic preview that keeps the shopper and garment visually consistent.

Step 04

Shopper checks out confidently

With less uncertainty around the look, shoppers can make faster and more confident purchase decisions.

Overview

FitLayer provides a hosted widget, JavaScript SDK, and REST API for adding AI-powered virtual try-on to fashion commerce experiences.

Use publishable keys in the browser. Keep secret keys on a trusted server.

Install the SDK

Install the browser package using your preferred package manager.

Terminal
npm install @fitlayer/widget

Authentication

Publishable keys initialize storefront widgets. Secret keys authenticate server-side API operations.

Initialize a product

JavaScript
FitLayer.init({
  apiKey: 'pk_live_...',
  productId: 'coat-248',
  mountId: 'try-on'
});

Listen for events

Subscribe to widget lifecycle events to connect analytics and storefront behavior.

Create a try-on

Submit a person image and garment identifier. The response returns an asynchronous job ID.

Error handling

API responses include stable error codes, readable messages, and request IDs for support.
