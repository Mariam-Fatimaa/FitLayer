Quick start

Add FitLayer to an existing product page without restructuring the storefront.

Terminal 
npm install @fitlayer/widget
product-page.ts 
import { FitLayer } from '@fitlayer/widget';

FitLayer.init({
  apiKey: 'pk_live_...',
  productId: 'coat-248',
  mountId: 'fitlayer-button',
  theme: 'system'
});
